📌 核心文件说明：

【Infrastructure Layer - 基础设施层】
  ├─ DatabaseManager.h/cpp     → MySQL连接管理、SQL执行
  └─ TableManager.h/cpp        → 表级CRUD操作封装

【Core/DAO Layer - 核心数据访问层】⭐ 最重要
  ├─ entities/                 → User, Order, Product 业务实体
  ├─ dao/                      → UserDAO, OrderDAO, ProductDAO 数据访问
  └─ DatabaseService.h/cpp     → 单例，统一管理连接和DAO

【Business Logic Layer - 业务逻辑层】
  ├─ user_module/              → 用户相关业务（注册、查询、更新等）
  └─ order_module/             → 订单相关业务（创建、完成、查询等）

【Application Layer - 应用层】
  └─ app/main.cpp              → 程序入口，演示完整使用流程
